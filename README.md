# Flappy Bird - Unity 2D
Modified version of Flappy Bird for DES643 Design for AR/VR Lab Assignment.

## Modification: Context-Aware Sound Design
- Score sound plays every time the score increases
- Hit sound plays on collision with pipes or ground
- Sounds don't overlap (using PlayOneShot + lock flag)
- Game Over screen shows final score and student credit

## How to Run
1. Clone this repo
2. Open in Unity 2022.3.29f1 or later
3. Open Assets > Scenes > Flappy Bird
4. Press Play
